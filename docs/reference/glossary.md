# 🚀 Glossary

- Artifact: job-linked document identity (`resume` or `cover_letter`).
- Artifact version: immutable content snapshot for an artifact.
- Active version: pointer to current version shown in editor.
- Swarm run: one optimization execution over an artifact.
- Legal move: bounded deterministic edit operation.
- Out-of-region edit: attempted edit outside editable policy zones.
- Compile regression: before-compiles, after-fails case.
- Evidence Vault: user-provided grounding assets.
- `supported_by`: citation IDs attached to claim-introducing edits.
